# Spacing Tokens — 8px Grid

All spacing must be a multiple of 4px. Prefer multiples of 8px.

| Token | Value | Common Use |
|-------|-------|-----------|
| `--space-xs` | 4px | Icon-to-label gap, tight row padding |
| `--space-sm` | 8px | Between tightly related elements (label → input) |
| `--space-md` | 16px | Card inner padding, gap between form fields |
| `--space-lg` | 24px | Between major card sections, option list gap |
| `--space-xl` | 32px | Section separation, hiw-badge margin-top |
| `--space-2xl` | 40px | Progress bar bottom margin, modal section gap |
| `--space-3xl` | 64px | Modal padding (top/bottom) |

## Layout Constants
| Rule | Value |
|------|-------|
| Max content width | `560px` |
| Page horizontal padding | `24px` |
| Content padding-top | `20px` |
| Content padding-bottom | `80px` |
| Top bar padding | `20px 40px` |
| Content stack gap | `24px` |
