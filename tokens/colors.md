# Color Tokens

## Primary Palette
| Token | Hex | Usage |
|-------|-----|-------|
| Primary | `#056257` | CTAs, active states, progress, links, checkmarks |
| Primary Light | `#f5fffe` | Selected tile tint, modal bg, input focus background |
| Primary Dark | `#044d44` | Button hover state |

## Neutral Palette
| Token | Hex | Usage |
|-------|-----|-------|
| Text Primary | `#272727` | All headings, body copy, input values |
| Text Muted | `#7e7e7e` | Descriptions, step counters, placeholders, sub-labels |
| Border | `#d4d4d4` | Input borders, dividers, progress track, unselected tiles |
| Surface | `#ffffff` | Page background, card background |
| Overlay | `rgba(0,0,0,0.5)` | Modal backdrop (always paired with `backdrop-filter:blur(10px)`) |

## Semantic
| Token | Hex | Usage |
|-------|-----|-------|
| Success | `#056257` | Completed states, checkmarks (same as primary) |
| Error | `#d93025` | Validation errors, alert states |
| White | `#ffffff` | Text on primary buttons, inverse text |
